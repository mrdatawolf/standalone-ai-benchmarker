Share Results

AI Bench can sync your benchmark results to a shared Google Sheet, so you can compare your hardware against other machines on a leaderboard.

This step requires a one-time Google Cloud credential setup. It is completely optional — all results are saved locally regardless.

Follow these 5 steps to get your credentials:

1
Open Google Cloud Console

console.cloud.google.com ↗ — sign in with any Google account.

2
Create or select a project

Click the project dropdown at the top → "New Project". Name it anything (e.g. "ai-bench").

3
Enable the Google Sheets API

Go to APIs & Services → Library → search "Google Sheets API" → Enable.

4
Create OAuth credentials

Go to APIs & Services → Credentials → Create Credentials → OAuth client ID. Set Application type to "Desktop app". Click Create.

5
Paste your credentials below

Copy the Client ID and Client Secret from the dialog that appears.
